# Restaurant Management System

A Java and JavaFX based Restaurant Management System using Object-Oriented Programming concepts.

## Features
- Customer registration
- Table reservation
- Food ordering
- Bill calculation
- JavaFX GUI
- Basic error handling

## OOP Concepts Used
- Classes and Objects
- Encapsulation
- Inheritance
- Aggregation
- Exception Handling
- File Handling

## Technologies Used
- Java
- JavaFX
- NetBeans / Maven

## Project Description
This project is designed to manage basic restaurant operations. It allows customers to reserve tables, place food orders, and calculate bills through a simple JavaFX GUI.

## Project Status
Completed as an academic OOP project.
